# PPTX Skill

Create and modify PowerPoint presentations with slides, layouts, speaker notes, templates, and animations.

## Core Capabilities

- **Create presentations**: Generate `.pptx` files with title, content, two-column, and blank-layout slides.
- **Edit slides**: Add/remove slides, reorder, update text and images in existing files.
- **Speaker notes**: Attach structured notes to every slide for presenter guidance.
- **Charts**: Insert bar, line, pie, and scatter charts from tabular data.
- **Images**: Embed photos, diagrams, and screenshots with precise placement.
- **Transitions/Animations**: Apply built-in slide transitions and object animation sequences.
- **Templates**: Apply a master `.pptx` template for brand-consistent decks.

## Creating a Presentation

```python
def create_deck(title: str, slides: list[dict]) -> bytes:
    """
    slides: [{"heading": str, "body": str, "notes": str, "bullets": [...]}]
    """
    import io
    prs = Presentation()
    slide_width = prs.slide_width
    slide_height = prs.slide_height

    # Title slide
    layout = prs.slide_layouts[0]  # Title Slide
    slide = prs.slides.add_slide(layout)
    slide.shapes.title.text = title
    slide.placeholders[1].text = "Generated by Pakalon"

    for s in slides:
        layout = prs.slide_layouts[1]  # Title and Content
        slide = prs.slides.add_slide(layout)
        slide.shapes.title.text = s.get("heading", "")

        body = slide.placeholders[1]
        tf = body.text_frame
        tf.clear()

        if s.get("bullets"):
            for i, bullet in enumerate(s["bullets"]):
                para = tf.add_paragraph() if i else tf.paragraphs[0]
                para.text = bullet
                para.level = 0
        elif s.get("body"):
            tf.paragraphs[0].text = s["body"]

        if s.get("notes"):
            notes_slide = slide.notes_slide
            notes_slide.notes_text_frame.text = s["notes"]

    buf = io.BytesIO()
    prs.save(buf)
    return buf.getvalue()
```

## Adding a Chart

```python
def add_chart_slide(prs: Presentation, title: str, categories: list, series: dict) -> None:
    """
    series: {"Revenue": [100, 200, 300], "Costs": [80, 150, 220]}
    """
    from pptx.util import Inches
    slide = prs.slides.add_slide(prs.slide_layouts[5])  # Blank
    slide.shapes.title  # may be None on blank layout

    chart_data = ChartData()
    chart_data.categories = categories
    for name, values in series.items():
        chart_data.add_series(name, values)

    chart = slide.shapes.add_chart(
        XL_CHART_TYPE.BAR_CLUSTERED,
        Inches(1), Inches(1.5), Inches(8), Inches(5),
        chart_data,
    ).chart

    chart.has_title = True
    chart.chart_title.text_frame.text = title
```

## Applying a Brand Template

```python
def apply_template(content_slides: list[dict], template_path: str) -> bytes:
    import io, copy
    prs = Presentation(template_path)
    # Remove existing placeholder slides (keep masters)
    xml_slides = prs.slides._sldIdLst
    for slide in list(prs.slides):
        ref = slide._element
        xml_slides.remove(ref)

    for s in content_slides:
        layout = prs.slide_layouts[1]
        slide = prs.slides.add_slide(layout)
        slide.shapes.title.text = s.get("heading", "")
        slide.placeholders[1].text_frame.paragraphs[0].text = s.get("body", "")

    buf = io.BytesIO()
    prs.save(buf)
    return buf.getvalue()
```

## Setting Slide Transition

```python
from pptx.oxml.ns import qn
from pptx.oxml import parse_xml

def set_fade_transition(slide) -> None:
    """Apply a fade transition to a slide element."""
    transition_xml = '<p:transition xmlns:p="http://schemas.openxmlformats.org/presentationml/2006/main"><p:fade/></p:transition>'
    slide._element.insert(2, parse_xml(transition_xml))
```

## Best Practices

- Always use `Inches()` or `Pt()` units — raw pixel math is fragile across screen sizes.
- Limit bullet points to 5–6 per slide; prefer visual hierarchy over wall-of-text.
- Speaker notes should be complete sentences — they must work as a standalone script.
- When applying templates, preserve the slide master and layouts; never recreate them.
- Prefer `slide_layouts[1]` (Title + Content) for standard slides, `slide_layouts[5]` (Blank) for charts/images.
- For large decks (>30 slides), batch-generate and merge sections rather than one monolithic run.
