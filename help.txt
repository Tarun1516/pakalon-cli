Usage: oh-my-logo [options] [text] [palette]

Display giant ASCII art logos with colorful gradients in your terminal

Arguments:
  text                       Text to display (use "\n" for newlines or "-" for
                             stdin)
  palette                    Color palette to use (default: "grad-blue")

Options:
  -V, --version              output the version number
  --palette-colors <colors>  Custom colors as JSON array or comma-separated
                             list
  -f, --font <name>          Figlet font name (default: "Standard")
  -l, --list-palettes        List available palettes
  --gallery                  Render text in all available palettes
  --color                    Force color output even in pipes
  --no-color                 Disable color output
  -d, --direction <dir>      Gradient direction: horizontal, vertical, or
                             diagonal (default: "vertical")
  --filled                   Use filled characters instead of outlined ASCII
                             art
  --block-font <font>        Font for filled mode (3d, block, chrome, grid,
                             huge, pallet, shade, simple, simple3d,
                             simpleBlock, slick, tiny) (default: "block")
  --letter-spacing <number>  Letter spacing for filled mode
  --reverse-gradient         Reverse gradient colors
  -h, --help                 display help for command
